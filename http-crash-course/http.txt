# WHAT ARE HTTP HEADERS
http header let the client and the server pass additional information with an HTTP request or response.
An HTTP header consists of its case-insensitive name followed by a colon (:), then by its value. Whitespace before the value is ignored.

metadata(data about data) ---------------> key value  sent along req and response
